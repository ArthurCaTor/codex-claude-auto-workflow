# State Machine

Allowed states:

```text
IDLE
READY_FOR_CLAUDE
CLAUDE_RUNNING
READY_FOR_CODEX_REVIEW
CODEX_REVIEWING
ACCEPTED
NEEDS_FIX
BLOCKED
OWNER_DECISION_REQUIRED
OWNER_REVIEW_REQUIRED
ERROR
```

Normal path:

```text
IDLE
-> READY_FOR_CLAUDE
-> CLAUDE_RUNNING
-> READY_FOR_CODEX_REVIEW
-> CODEX_REVIEWING
-> ACCEPTED
-> OWNER_REVIEW_REQUIRED
```

Optional report-only fix path:

```text
CODEX_REVIEWING
-> READY_FOR_CLAUDE  (same task, report-only fix, budgeted)
-> CLAUDE_RUNNING
-> READY_FOR_CODEX_REVIEW
```

Manual recovery states:

```text
OWNER_REVIEW_REQUIRED
OWNER_DECISION_REQUIRED
BLOCKED
ERROR
```

Only the human owner exits manual recovery states.
